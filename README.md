# Описание
Дзен игра, в которой необходимо ходить по бесконечному лабиринту. Запускается и отображается прямо в терминале. Отображает коридоры в стиле старого wolfenstein - в псевдо 3d
# Запуск
В корне проекта
```
cargo run --release
```
